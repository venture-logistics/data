os Error Last updated: 1oth June 2025

If you're here you've received an os error!

NameError: name 'os' is not defined

Which is a pain!

The Fix

Visit the following folder; - or thereabouts, but it'll be very similar.

C:\Users\artic\AppData\Local\Packages\PythonSoftwareFoundation.Python.3.11_qbz5n2kfra8p0\LocalCache\local-packages\Python311\Scripts

Once there, copy the path, and in the Windows Search (botom left), type (copy and paste);

Edit environment variables

Open this program, click "Environment Variables," normally at the bottom, above "Cancel," this will open a new dialoge box.

Add new for both, i.e. you and system variables, add a name and paste in the path.

Hit OK and OK again.

In VS Code close the terminal window and open another.

CD back to the chat-bot-main folder, for example;

cd desktop
